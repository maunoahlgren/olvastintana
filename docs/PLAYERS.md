## Ability Types
| Symbol | Type | Description |
|--------|------|-------------|
| 💥 | boost | Stat boost, conditional or random |
| 🎲 | chaos | Unpredictable — could help or hurt |
| ⚡ | reactive | Triggers after opponent reveals card |
| 🔒 | restriction | Limits what opponent can do next |
| 🏆 | dominant | Wins all ties regardless of stats |
